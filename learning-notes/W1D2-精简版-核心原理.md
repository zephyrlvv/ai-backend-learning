## 🎯 今天必须理解的 3 个核心概念

### 概念 1：FastAPI 是干什么的？

**一句话**：它帮你把 Python 函数变成 HTTP 接口，让前端能调用。

**类比**：
- Vue 里你写 `methods`，用户点击按钮调用
- FastAPI 里你写函数，前端通过 URL 调用

**代码对比**：

```javascript
// Vue - 前端函数，用户点击触发
methods: {
  hello() {
    return "Hello"
  }
}
```

```python
# FastAPI - 后端函数，前端通过 HTTP 调用
@app.get("/")  # 访问 http://localhost:8000/ 就会执行这个函数
def hello():
    return {"message": "Hello"}
```

**就业价值**：
> 面试问："Python 怎么写 API？" 
> 答：用 FastAPI，用装饰器把函数变成接口，自动处理 HTTP 请求和响应。

### 概念 2：什么是 CORS？（今天最重点）

**一句话**：浏览器不让网页随便访问其他域名，CORS 就是"申请通行证"。

**场景**：
- Vue 跑在 `http://localhost:5173`
- Python 跑在 `http://localhost:8000`
- 端口不同 = 浏览器认为是"不同域名" = 拦截

**类比 Vue Router 导航守卫**：
```javascript
// Vue Router - 守卫检查用户权限
router.beforeEach((to, from, next) => {
  if (允许访问) next()
  else 阻止
})
```

```python
# FastAPI CORS - 守卫检查请求来源
app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:5173"],  # 只允许这个域名
    # 其他的都拦下
)
```

**关键理解**：
- CORS 是**后端配置**的（告诉浏览器"我允许谁访问我"）
- 不是前端能解决的（前端想访问，但浏览器不让）

**面试价值**：
> 面试问："前后端分离项目遇到跨域怎么办？"
> 答：后端配置 CORS，设置 Access-Control-Allow-Origin 响应头，允许前端域名访问。

### 概念 3：GET 请求参数怎么传？

**一句话**：URL 后面 `?key=value` 就是 GET 参数。

**完整流程**：

```
浏览器地址栏：
http://localhost:8000/chat?msg=你好
                    │      │   │
                    │      │   └── 参数值
                    │      └────── 参数名
                    └───────────── 路由路径
```

**前端（Vue）怎么发**：
```javascript
// 方式 1：直接拼 URL（不推荐）
axios.get('http://localhost:8000/chat?msg=' + message)

// 方式 2：用 params（推荐，会自动编码）
axios.get('http://localhost:8000/chat', {
  params: { msg: message }  // 自动变成 ?msg=xxx
})
```

**后端（Python）怎么收**：
```python
@app.get("/chat")
def chat(msg: str):  # FastAPI 自动把 URL 参数 msg 传进来
    return {"reply": f"收到：{msg}"}
```

**类比 TypeScript 类型定义**：
```typescript
// TS 定义函数参数类型
function chat(msg: string): string {
  return `收到：${msg}`
}
```

```python
# Python 3.6+ 也有类型提示（FastAPI 强制要求）
def chat(msg: str) -> dict:  # msg 必须是字符串，返回字典
    return {"reply": f"收到：{msg}"}
```

### 步骤 2：修改 App.vue（10 分钟）

直接覆盖 `src/App.vue`：

```vue
<template>
  <div style="padding: 20px;">
    <h1>Vue + Python 联调测试</h1>
    <input v-model="msg" placeholder="输入消息" />
    <button @click="send">发送给 Python</button>
    <p v-if="result">返回结果：{{ result }}</p>
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue'
import axios from 'axios'

const msg = ref('')
const result = ref('')

const send = async () => {
  try {
    const res = await axios.get('http://localhost:8000/chat', {
      params: { msg: msg.value }
    })
    result.value = res.data.reply
  } catch (e) {
    result.value = '错误：' + (e as Error).message
  }
}
</script>
```

### 步骤 3：Python 加 CORS（10 分钟）

修改 `G:\kimiProject-pc\main.py`：

```python
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware  # 新增

app = FastAPI()

# 新增：允许 Vue 访问
app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:5173"],
    allow_methods=["*"],
    allow_headers=["*"],
)

@app.get("/")
def hello():
    return {"message": "Hello"}

@app.get("/chat")
def chat(msg: str):
    return {"reply": f"Python 收到：{msg}"}
```

## ❓ 如果报错

| 错误 | 原因 | 解决 |
|------|------|------|
| Network Error | Python 没启动 或 CORS 没配置 | 检查 Python 是否在运行，重启 uvicorn |
| CORS policy | allow_origins 写错了 | 确认是 `http://localhost:5173`，不是 https |
| 404 Not Found | 路由写错了 | 检查 `@app.get("/chat")` 和 axios URL 是否一致 |
